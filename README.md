DeskFlow
Интеллектуальное приложение для организации рабочего пространства Windows, которое автоматизирует сортировку файлов, управление профилями рабочего стола, предоставляет виджеты и инструменты персонализации интерфейса.
Описание
DeskFlow — это комплексное решение для повышения продуктивности и организации рабочего пространства пользователей Windows. Приложение помогает автоматически сортировать файлы на рабочем столе, управлять различными профилями оформления, использовать полезные виджеты и синхронизировать настройки между устройствами.
Проект разработан в соответствии с требованиями спецификации DeskFlow версии 1.0 и реализует основные функции организации файловой системы, управления профилями пользователя и персонализации рабочей среды.

Установка из релиза

Скачайте последнюю версию из раздела Releases
Распакуйте архив в любую папку
Запустите DeskFlow.exe

При первом запуске приложение:

Создаст папку конфигурации в %AppData%/DeskFlow
Выполнит сканирование рабочего стола
Создаст начальную базу данных

Использование
Первый запуск

При запуске приложение автоматически сканирует рабочий стол
Все найденные файлы отображаются в разделе "Рабочий стол"
Файлы автоматически категоризированы по типам

Сортировка файлов
Виртуальная сортировка:

Откройте вкладку "Рабочий стол"
Нажмите "Сортировать виртуально"
Файлы будут упорядочены по категориям в списке

Физическое перемещение:

Перейдите на вкладку "Сортировка"
Выберите целевую папку (по умолчанию - рабочий стол)
Вернитесь на вкладку "Рабочий стол"
Нажмите "Переместить в папки"
Подтвердите операцию

Файлы будут перемещены в подпапки согласно их категориям.
Работа с профилями
Создание профиля:

Откройте вкладку "Профили"
Нажмите "Создать профиль"
Введите имя профиля
Текущее состояние рабочего стола будет сохранено

Переключение профиля:

Откройте список профилей
Нажмите "Применить" на нужном профиле
Состояние рабочего стола изменится согласно профилю

Использование виджетов
Активация виджета:

Откройте вкладку "Виджеты"
Выберите нужный виджет (Часы, Календарь, Заметки, Задачи)
Нажмите "Показать на рабочем столе"
Виджет появится в отдельном окне

Настройка виджетов:

Виджеты можно перемещать по экрану
Изменения в виджетах автоматически синхронизируются с главным окном
Закрытие виджета не удаляет данные

Мониторинг в реальном времени

Откройте вкладку "Сортировка"
Включите опцию "Мониторить рабочий стол в реальном времени"
Приложение будет отслеживать все изменения на рабочем столе
Опционально включите "Автоматическая виртуальная сортировка"

Резервное копирование
Экспорт данных:

Откройте вкладку "Настройки"
Нажмите "Экспорт данных"
Выберите место сохранения JSON файла
Все данные будут сохранены в один файл

Импорт данных:

Откройте вкладку "Настройки"
Нажмите "Импорт данных"
Выберите ранее сохраненный JSON файл
Данные будут загружены и объединены с текущими

Конфигурация
Файлы конфигурации
Приложение хранит данные в следующих файлах:
%AppData%/DeskFlow/
├── data.json          # Основные данные (файлы, профили, задачи)
└── notes.txt          # Текст заметок


Устранение неполадок
Приложение не запускается
Проблема: Ошибка "Framework not found"
Решение: Установите .NET 7.0 Runtime с официального сайта Microsoft
Проблема: Ошибка "Access denied"
Решение: Убедитесь, что у вашего пользователя есть права на запись в папку %AppData%
Мониторинг не работает
Проблема: Файлы на рабочем столе не отслеживаются
Решение:

Проверьте, включен ли мониторинг в настройках
Перезапустите приложение
Проверьте права доступа к рабочему столу

Файлы не перемещаются
Проблема: Ошибка при перемещении файлов
Решение:

Убедитесь, что файлы не используются другими программами
Проверьте наличие свободного места на диске
Проверьте права на запись в целевую папку

Виджеты не отображаются
Проблема: Виджет не появляется на рабочем столе
Решение:

Проверьте, не находится ли виджет за другими окнами
Закройте и откройте виджет снова
Перезапустите приложение
